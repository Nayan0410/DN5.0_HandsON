# Digital Nurture 5.0 (Java FSE)

This repository contains my solutions, assignments, and hands-on exercises completed as part of the **Cognizant Digital
Nurture 5.0 - Java Full Stack Engineer (Java FSE)** learning program.

The repository is organized by week and technology, following a structured and industry-standard approach to learning and implementation.

---

## Learning Journey

The Cognizant Digital Nurture 5.0 – Java FSE program is designed to build a strong foundation in software engineering by covering engineering concepts, programming languages, modern frameworks, and deployment platforms.

The learning path includes:

### Engineering Concepts

- Design Patterns and Principles
- Data Structures and Algorithms

### Programming Languages

- Java
- SQL (Basic & Advanced)
- Test-Driven Development (JUnit 5 & Mockito)
- SLF4J Logging Framework

### Products & Frameworks

- Spring Core & Maven
- Spring Data JPA
- Hibernate
- Spring Boot
- Spring REST
- Microservices with Spring Cloud
- React

### Platforms & DevOps

- Git
- CI/CD
- Docker
- Cloud Fundamentals
